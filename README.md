# vpyramid
vpyramid is a simple and fun script that displays vertical pyramid ASCII art on your linux terminal using randomly generated characters.

## Installation
1. Using Makefile:

`make build`

2. Using shell

`chmod +x install.sh`

`./install.sh`

## Usage
`vpyramid <max_length>`

`max_length` can be any number between 5 - 15.

## Uninstallation
`vpyramid remove`
